Istruzioni:

- Clonare il repository (test_frontend). Aprire un terminale ed eseguire il comando "git clone https://github.com/pep1994/test_frontend.git"
- Una volta clonato il repository ci saranno due cartelle all'interno, una chiamata test_frontend_ex1 e l'altra test_frontend_ex2.

Istruzioni per ex1:

- Aprire un terminale ed entrare nella cartella "test_frontend_ex1"
- Lanciare il comando "npm install"
- Aprire nel browser il file index.html situato dentro la cartella dist sempre all'interno della directory test_frontend_ex1

Istruzioni per ex2:

- Aprire un terminale ed entrare nella cartella "test_frontend_ex2"
- Assicurarsi che nella macchina sia installato node.js e Vue; per verificare che node sia installato eseguire i comandi "node -v" e "npm -v". Se node non è installato, installarlo tramite il sito ufficiale. Per verificare che Vue CLI sia installato globalmente eseguire il comando "vue --version". Se Vue non è installato eseguire il seguente comando: "npm install -g @vue/cli" .
- All'interno della cartella "test_frontend_ex2" lanciare il comando "npm install".
- Sempre all'interno della directory "test_frontend_ex2" lanciare il comando "npm run serve" . Sul terminale verrà indicato un indirizzo IP locale (localhost). Navigare verso questo indirizzo con il Browser di preferenza e l'applicazione sarà visibile.
